# Fabric Events

Developer resources for building event-driven solutions with Microsoft Fabric — scenarios, design guides, recipes, and community discussion across Business Events, Fabric Events, and Azure Events.

**[→ Visit the site](https://microsoft.github.io/fabric-events/)**

---

## What's here

| Section | What you'll find |
|---|---|
| **Business Events** | End-to-end scenarios, design guides, and recipes for publishing and consuming Business Events across Notebooks, User Data Functions, Eventstream, Activator, and Eventhouse |
| **Fabric Events** | Platform-level events emitted by Fabric services — item lifecycle, workspace changes |
| **Azure Events** | Storage Account events flowing into Fabric via Eventstream |

## Contributing

This site grows with the community. If you have a scenario to share, a recipe to add, or a correction to make:

1. Fork this repo
2. Create a branch from `main`
3. Add or edit content under `docs/`
4. Open a pull request

See [INSTRUCTIONS.md](INSTRUCTIONS.md) for the full content guide, naming conventions, and deploy commands.
